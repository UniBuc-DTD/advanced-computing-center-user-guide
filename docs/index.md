# Advance Computing Center User Guide

Welcome to the documentation for the [University of Bucharest](https://unibuc.ro/)'s [Advanced Computing Center](https://edis.unibuc.ro/data-center-si-laboratoare-de-acces/) users. This site contains useful information for end users of the computing system (teachers, researchers and students).

If you're not yet using the ACC and want to find out how it can help you, check out our [getting started](getting-started.md) guide.

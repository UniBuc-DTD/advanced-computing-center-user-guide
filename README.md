# Documentație de utilizare Advanced Computing Center

## Scop

Acest repo conține documentația destinată utilizatorilor **Centrului de Calcul Avansat** (_Advanced Computing Center_) al [Universității din București](https://unibuc.ro/).

## Resurse disponibile

În funcție de sarcina computațională pe care doriți să o rulați, ACC-ul vă pune la dispoziție mai multe tipuri de hardware:

- Sistemul de calcul paralel de tip _High-Performance Computing_ (HPC-CPU), un cluster de SLURM cu noduri cu multe CPU-uri
- Nodul de calcul cu plăci video (HPC-GPU), destinat în special pentru aplicațiile de Inteligență Artificială (AI) și Învățare Automată (ML)
- Infrastructura de hiperconvergență (HCI), concepută pentru rularea aplicațiilor educaționale / de cercetare pe o platformă bazată pe mașini virtuale
